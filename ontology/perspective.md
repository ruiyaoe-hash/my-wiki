# Ontology Perspective

## 观察视角声明

本系统的 Ontology 以 **Runtime（执行引擎）** 为观察视角。

不以知识管理为视角，不以存储实现为视角，不以用户界面为视角。

## 这意味着什么

一个东西是不是"一等对象"，判断标准只有一个：

**Runtime 在执行过程中，需不需要找到它、知道它此刻的状态、并对它执行操作？**

需要 → 一等对象。
不需要 → 属性、标签、配置、或辅助设施。

## 适用范围

本文件为 Agent Runtime 项目的顶层设计文档。
Phase 0 锁定后不再修改（除非 Runtime 本身的定位发生质变）。
