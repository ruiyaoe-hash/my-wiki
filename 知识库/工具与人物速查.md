## 工具

### Obsidian
*基于本地 Markdown 文件的知识管理工具，支持双向链接和图谱视图。*

## 概述
Obsidian 是基于本地 Markdown 文件的知识管理工具，支持双向链接和图谱视图。
## 在 LLM Wiki 生态中的角色
- 知识库的"IDE"：浏览 wiki、查看图谱、发现关联
- 与 Claude Code / Codex 配合：AI 写 wiki，Obsidian 展示
- 纯 Markdown 无锁定，数据完全属于用户

## 概述

## OKF 的意义

- 让不同 AI 工具能交换编译后的知识

## 工具

### Dolt
*Git 风格的数据库版本控制工具，原生支持分支、提交、合并，兼容 MySQL 协议。*

## 概述
Dolt 是 Git 风格的数据库版本控制工具，开源 (23K Star, Apache-2.0)。
## 核心功能
- 原生支持分行、提交、合并
- 行级历史追踪（自动 dolt_history 视图）
- 兼容 MySQL 协议
- MCP Server 支持 AI Agent 在隔离分支操作
## 性能
- 读写接近 MySQL
- TPC-C 约为 MySQL 的 54%
- 超 1G 数据时性能下降
## 典型场景
- 数据科学团队的分支实验
- AI Agent 安全数据操作
- 多团队协作数据修改
