# AI Meets Brain (2025)

> 来源：ChatGPT 对话推荐（2026-07-06 会话）
> 地位：跨学科最强的一篇综述，引用 400 篇文献
> 评价：被 ChatGPT 评为"如果想做 AGI/Agent/Memory，这是必读"

## 核心贡献

这篇论文横跨认知神经科学与 AI 两个学科，建立了两者之间的深层对应关系：

- **神经科学发现** → Agent Architecture 设计原则
- **人脑记忆机制** → AI 记忆系统的工程实现
- **认知心理学模型** → Agent 决策和规划的架构参考

## 关键议题

1. 人脑的睡眠巩固机制如何启发 AI 的 offline consolidation
2. 海马体的记忆索引功能如何映射到 Agent 的检索系统
3. 前额叶的工作记忆管理如何指导上下文窗口的设计
4. 杏仁核的情绪标记如何影响记忆优先级排序
5. 默认模式网络的自我参照处理如何对应 Agent 的 Identity 和 Reflection

## 对 Wiki 的意义

当前 Wiki 在认知科学和 AI 之间已经有一些连接：
- [[FadeMem 生物启发遗忘]] 覆盖了遗忘机制的生物学映射
- [[HippoRAG 记忆框架]] 覆盖了海马体启发的检索

但缺少一篇从神经科学到 Agent Architecture 的系统性总览。这篇综述填补了这个缺口，也是未来产品化时"认知操作系统"理论框架的重要参考。
