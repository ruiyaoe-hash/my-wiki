# Memory in the Age of AI Agents (2025)

> 来源：ChatGPT 对话推荐（2026-07-06 会话）
> 地位：AI Memory 领域目前最重要的一篇综述，近 50 位作者合著
> 评价：被 ChatGPT 评为"未来几年引用率会非常高，Memory 领域目前最接近圣经的一篇"

## 核心贡献

这篇综述几乎覆盖了 AI Agent 记忆研究的所有维度：
- 所有 Memory 分类体系
- 所有 Agent Framework 中的记忆设计
- 所有 Benchmark 和评估方法
- 所有 Future Direction（未来研究方向）

## 覆盖维度

1. **记忆分类体系**：建立了从认知科学到 AI Agent 的完整分类映射
2. **框架对比**：横向对比了 MemGPT/Letta、Mem0、Zep、LangGraph、Cognee 等主流框架的记忆设计
3. **评估基准**：梳理了 LoCoMo、MemBench、MemoryAgentBench、MemoryArena 等评测体系
4. **开放问题**：识别了持续巩固、因果检索、可信反思、主动遗忘等未解决问题

## 对 Wiki 的意义

这篇论文的结构本身就是 wiki 外部记忆域可以参照的"全景骨架"。当前 [[外部记忆-行业全景]] 已经做了部分框架对比，但缺少系统性的分类体系和评估基准总结。这篇综述可以填补这个缺口。
