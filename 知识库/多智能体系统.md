## 概述
多智能体系统是由多个 AI Agent 协同完成任务的系统。与单 Agent 不同，多 Agent 需要解决协调、通信、状态共享等问题。

## 核心挑战
- **协调崩溃 (Coordination Collapse)**：Agent 间沟通失效，是最大单类失败来源，占故障的 36.9%（伯克利研究）
- **错误放大**：谷歌 DeepMind 发现，无结构的 Agent 网络将错误放大高达 17.2 倍
- **涌现行为**：LLM 驱动的 Agent 通过自然语言协调，产生未被预先设计的行为模式

## 失败率数据
- MetaGPT、AutoGen 等 5 个主流系统失败率从 41% 到 86.7% 不等
- 共整理出 14 种核心失败模式
