# From Human Memory to AI Memory (2025)

> 来源：ChatGPT 对话推荐（2026-07-06 会话）
> 地位：认知科学和 AI 结合最好的综述论文
> 评价：被 ChatGPT 评为"第一次系统回答人脑为什么这样记忆、AI 为什么应该这样记忆"

## 核心贡献

这篇论文最大的贡献是建立了从人类记忆系统到 AI Agent 记忆设计的完整映射。

人脑记忆的分类体系：
- 感觉记忆（Sensory Memory）→ AI 的输入缓冲
- 工作记忆（Working Memory）→ AI 的上下文窗口
- 情景记忆（Episodic Memory）→ AI 的会话历史和经历
- 语义记忆（Semantic Memory）→ AI 的知识库
- 程序性记忆（Procedural Memory）→ AI 的技能和操作流程
