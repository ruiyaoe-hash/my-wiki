## 核心工具栈
- **Obsidian**：本地 Markdown 知识库，支持图谱视图
- **Claude Code / Codex**：AI 编程助手，可自动维护 wiki
- **LLM Wiki 模式**：AI 写 wiki，人读 wiki 并提问

## 关键实践
1. 所有内容存为纯 Markdown，无锁定
2. AI 摄入原始资料 → 创建概念页、实体页、交叉引用
3. 定期运行健康检查（断链、矛盾、过时）
4. 用 hot.md 做热缓存加速重复查询
