# OpenViking: The Context Database for AI Agents

**仓库：** [volcengine/OpenViking](https://github.com/volcengine/OpenViking) · 字节跳动火山引擎 · AGPLv3

解决 Agent 开发五大痛点：上下文碎片化、需求暴涨、检索效果差、不可观测、记忆迭代弱。

核心创新：文件系统范式替代向量存储，L0/L1/L2 三层按需加载，目录递归检索，检索轨迹可视化，自动会话管理。

支持五种 VLM 后端：火山 Doubao / OpenAI / Codex / Kimi / GLM。
