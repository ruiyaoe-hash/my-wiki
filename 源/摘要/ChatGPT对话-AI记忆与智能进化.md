## 核心内容

本次对话分为三个主要段落：

**第一段：AI记忆第一性原理**
- 提出 Intelligence = Model × Memory × Reasoning × Action × Feedback
- 区分 Context（工作记忆） ≠ Memory（长期记忆）
- 阐明 Agent 循环：Observe → Think → Act → Reflect → Remember
- 引入"认知操作系统"八模块架构：LLM → Memory → Reasoning → Planning → Reflection → Identity → Goals → Values

**第二段：前沿知识推荐**
- 五篇必读论文、五个开源项目、八个研究热点（已全部入库 wiki）
- 提出了六大共识：Memory ≠ Context、Memory 是系统非单一体、Memory 有动态生命周期、Memory 从 RAG 独立、Memory 成为 Agent 第一公民、Memory 从数据到资产

**第三段：学习方法论**
- 八步前沿学习系统：知识地图 → 第一性原理 → 经典论文 → 概念图谱 → 论文循环 → 工程实践 → 个人知识库 → 原创理论 + 问题树
- 针对用户定制的六个月研究路线
