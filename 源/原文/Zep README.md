# Zep: Examples & Integrations (README)

> GitHub: https://github.com/getzep/zep
> 企业级Agent记忆平台

Zep 是托管式 Agent 记忆平台。此仓库包含示例代码、框架集成和工具。

核心技术栈: Graphiti（开源时序知识图谱框架）作为底层引擎。

支持的框架集成: Google ADK, Microsoft Agent Framework, AutoGen, AG2, CrewAI, LangGraph, LiveKit, Pydantic AI, Mastra, Vercel AI SDK

提供 Benchmark 能力: LoCoMo, LongMemEval 记忆评测
